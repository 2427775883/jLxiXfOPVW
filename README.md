# Python学生成绩管理系统

## 核心功能
- 学校管理
- 班级管理
- 课程管理
- 年级管理
- 用户管理
- 权限管理
- 角色管理
- 日志管理
- 人员管理

## 适用场景
- 学校内部学生成绩管理
- 教育机构成绩记录
- 教学辅助系统

## 技术特点
- 后端：Python、Django
- 前端：HTML、CSS、JavaScript、jQuery
- 数据库：MySQL
- 开发工具：Pycharm、Navicat
- 环境要求：Python 3.0以上、MySQL 8.0
- 可运行性：代码可运行，根据步骤配置环境即可
- 远程调试：可提供远程调试服务
